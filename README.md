# README

## インストール
'''
$ git clone git@github.com:IJGN/LifeLabel.git
'''
'''
$ bundle install --path vendor/bundle
'''
'''
$ bin/rails db:create db:migrate
'''
'''
$ bin/rails db:seed:test
'''
'''
$ bin/rails db:seed:master
'''
'''
$ bin/rails s
'''
